# **RU:**
Плагин, добавляющий механику шахт, привязанных к региону. Благодаря ему, вы можете настроить расписание открытия шахт, блоки, которые можно в них ломать, а также лут, который будет выпадать при поломке того или иного блока шахты.

# **EN:**
A plugin that adds a mine mechanic tied to regions. With it, you can set up mine opening schedules, which blocks can be broken inside them, and the loot that will drop when breaking specific mine blocks.
